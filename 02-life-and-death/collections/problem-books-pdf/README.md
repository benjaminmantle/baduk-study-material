# Problem books (PDF) — named masters & classics

Printable diagram booklets, rebuilt by the community from published CN/KR/JP problem books.
**⚠ community-grey** (positions freely circulated; underlying books are commercial) — see
[provenance](../../../docs/provenance.md). Level mostly SDK–Dan.

- **Composed masters:** Maeda Tsumego ×3, Go Seigen Tsumego Collection ×6, Hashimoto (Famous
  Creations, Masterpiece Collection), Kada Katsuji ×3, Fujisawa Shuko Classroom, Ishida Yoshio
  Challenge, Beautiful Dan-Level Tsumego 100, Showa Tsumego, Hitomi Tsumego.
- **Dictionaries:** Segoe Tsumego Dictionary ×4, Nie Weiping Tsumego Encyclopedia,
  Lee Chang-ho Selected Tsumego ×3.
- **Classic:** Heavenly Dragons (Tianlong Tu) ×4.

Source: 101books.github.io. Prefer playable answers? Use [`../../graded/gogameguru-weekly/`](../../graded/gogameguru-weekly/).
