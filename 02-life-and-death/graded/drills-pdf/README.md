# Graded practice drills (PDF)

High-volume rep practice, sorted by level. Community rebuilds (**⚠ grey**; 101books.github.io).

- **Beginner (30k–15k):** 200 Basic Go Problems, Compact Go Puzzles, Go Puzzles at a Glance,
  Bedtime Tsumego ×3, Weiqi L&D Drills — Beginner ×4.
- **DDK–SDK:** Weiqi L&D Drills — Intermediate ×4, Weiqi Rapid Drill 800 ×4, Ohashi-style
  Power-Up ×2, Common Life and Death Problems, 200 Selected (repetition), Weiqi Small Leisure,
  Seedling Project.
- **Progressive (graded 1→6):** Specialized Training in Tsumego 1–6, Weiqi L&D 1000 Problems ×4.

Do a set daily at a level where you go *fast*; re-solve until instant.
