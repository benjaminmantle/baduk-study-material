# 03 · Tesuji

The **vocabulary of skillful tactical moves** — the single biggest difference-maker on the road
from SDK to dan. Learn the recurring patterns, then drill them until you spot them instantly.

Core patterns to know: cutting tesuji, the **net (geta)** and **ladder (shichō)**, **clamp**,
**wedge**, **placement (tsuke/oki)**, **squeeze**, **snapback**, **under-the-stones**, and
liberty-filling techniques for capturing races.
