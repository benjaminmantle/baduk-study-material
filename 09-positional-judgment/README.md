# 09 · Positional Judgment

Reading the **whole board**: counting territory, valuing thickness, judging who's ahead, and
deciding what the position *needs* next. The dan-level skill that turns good tactics into won games.

**The four questions to ask, every game:**
1. **Who's ahead, and by how much?** Count each side's rough territory + prisoners + komi. You don't
   need to be exact — ±10 points is enough to steer by.
2. **What is thickness actually worth here?** A wall is worthless facing your opponent's stones and
   huge facing open space. Don't count it as territory — count what it *does*.
3. **What does the position need?** Ahead → simplify and take the cash. Behind → complicate, invade,
   or start a fight while the board is still open.
4. **Which move fixes the most?** Favour the biggest *dual-purpose* point — one that attacks *and*
   builds.

Practise by pausing a [pro game](../10-whole-games/) and estimating the score before you play on.
Reference: Cho Chikun's *Positional Judgment* (see [recommended-reading](../docs/recommended-reading.md)).
