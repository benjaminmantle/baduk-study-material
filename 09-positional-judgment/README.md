# 09 · Positional Judgment

Reading the **whole board**: counting territory, valuing thickness and influence, judging who's
ahead, and deciding what the position *needs* next (cash, attack, reduce, or simplify). A dan-level
skill that turns tactics into wins.

*Being filled.* The reference text is Cho Chikun's *Positional Judgment*; see
[recommended-reading](../docs/recommended-reading.md).
