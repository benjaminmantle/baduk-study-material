# Classic-era game records

Career archives of the old masters as compressed SGF packs. **Public domain** — game records are
facts. Extract with `tar -xzf name.tgz` (or open `.7z` with 7-Zip).

| Archive | Player / era |
|---|---|
| `shusaku.tgz` | Honinbo Shusaku ⭐ — the model for opening sense |
| `dosaku.tgz` | Honinbo Dosaku ⭐ — centuries ahead of his time |
| `go-seigen.tgz` | Go Seigen ⭐ — reshaped modern Go |
| `jowa.tgz` | Honinbo Jowa |
| `shusai.tgz` | Honinbo Shusai — the last hereditary Honinbo |
| `old-honinbo-house.tgz` | old Honinbo house / castle-games lineage |
| `takagawa.tgz` | Takagawa Kaku |
| `cho-chikun.7z` | Cho Chikun |
| `nakamura-sumire.tgz` | Nakamura Sumire (modern child prodigy) |

Source: the CWI/Brouwer public archive.
