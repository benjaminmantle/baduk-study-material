# 13 · AI & Tools

Free software that changed how everyone studies. You don't need a supercomputer — a laptop is fine.

| Tool | What it's for | Link |
|---|---|---|
| **KataGo** | The strongest open-source engine; the analysis brain behind the GUIs. | github.com/lightvector/KataGo |
| **KaTrain** ⭐ | The friendliest way to review a game with KataGo — shows point loss per move and your mistakes. **Best place to start.** | github.com/sanderland/katrain |
| **Lizzie / Lizzie-Yzy** | Classic live-analysis GUI. | github.com/featurecat/lizzie |
| **Sabaki** | Clean SGF editor + analysis frontend. | sabaki.yichuanshen.de |
| **OGS** | Free online play, puzzles, and a big SGF library. | online-go.com |

**How to actually use it to improve:** play a serious game → load it into KaTrain → don't obsess
over single blunders; look for your **recurring** mistakes (same kind of move, same phase) → drill
that topic using the matching folder here. AI tells you *where* you went wrong; these folders help
you fix *why*.
