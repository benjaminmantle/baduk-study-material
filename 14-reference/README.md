# 14 · Reference

The lookup shelf.

- **[Cross-language glossary](../docs/glossary.md)** — Go terms in English · Japanese · Korean ·
  Chinese, so you can study and search in any of them.
- **Ranks:** kyu (student, counts *down* 30k→1k) then dan (master, 1d→7d amateur; pro dan is a
  separate scale). See [study-paths](../docs/study-paths.md) for what to study at each.
- **Rules variants:** Japanese (territory) vs. Chinese (area) scoring, plus AGA and Ing — they
  almost always give the same winner; the main practical difference is how you count and handle
  dame/komi. Primers being added.
